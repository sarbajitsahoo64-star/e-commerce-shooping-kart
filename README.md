# ShopKart - Premium E-Commerce Storefront

A modern, fast, and responsive E-Commerce shopping cart application. The project features a premium "Pro" UI design with glassmorphic elements, smooth hover animations, and a seamless user experience. 

Based on a professional Figma design, the repository is structured as a monorepo containing both the frontend application and backend (Supabase) configurations.

## ✨ Features

- **Premium UI/UX:** Glassmorphic navigation, modern pill-shaped categories, and soft dynamic shadows.
- **Shopping Cart:** Add/remove items, adjust quantities, and calculate totals in real-time.
- **Product Filtering:** Filter items seamlessly by category and search queries.
- **Wishlist functionality:** Save your favorite items.
- **Order Management:** Dedicated dashboard to view past orders and their status.
- **User Authentication:** Beautifully designed login and sign-up modals.
- **Dark/Light Mode Ready:** Fully configured with a premium tailwind color palette.

## 💻 Tech Stack

- **Frontend:** React (Vite), TypeScript, Tailwind CSS
- **UI Components:** Lucide Icons, Radix UI primitives
- **Backend:** Supabase configurations
- **Deployment:** Ready for Vercel / Netlify
